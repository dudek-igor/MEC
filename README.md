## Tutorial

#### Requirements

- git
- docker

#### Run app localy via docker-compose

1. clone repo - `git clone https://github.com/dudek-igor/MEC.git`
2. change directory - `cd MEC`
3. Stop all your containers
4. type - `npm run docker`
5. Enjoy :)
